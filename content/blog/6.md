可恶的 GFW，你永远不知道他什么时候给你开个深度包检测，所有出口流量限速 64 KiB/s。所以我们要配置一个 ~~上海交大~~ 中科大 的 Flathub 国内镜像，避免防火墙给我们添麻烦。

```bash
# 修改系统级源（需要 sudo） 
sudo flatpak remote-modify flathub --url=https://mirrors.ustc.edu.cn/flathub/
# 修改用户级源（不需要 sudo） 
flatpak remote-modify --user flathub --url=https://mirrors.ustc.edu.cn/flathub/
```

> Gemini:
> 太棒了！中科大（USTC）的镜像站确实以稳定著称，很多时候交大镜像连接不畅，换它准没错。
