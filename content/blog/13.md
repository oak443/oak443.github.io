以下是安装流程。

## 1. 关闭 Secure Boot

在进行任何终端操作之前，必须重启电脑进入主板 BIOS，将 Secure Boot 选项设置为 Disabled 。

如果不关闭此选项，Linux 内核会拒绝加载本地编译的 NVIDIA 闭源驱动模块。

## 2. 安装所有驱动

在确认已关闭 Secure Boot 后，打开系统终端，执行安装命令为 Silverblue 叠加驱动：

```bash
rpm-ostree install akmod-nvidia xorg-x11-drv-nvidia-cuda
```

* `akmod-nvidia`：最核心的自动内核模块。它会在每次系统内核更新时，在后台自动把 NVIDIA 驱动编译匹配到新内核中，防止滚挂。

* `xorg-x11-drv-nvidia-cuda`：提供 CUDA、硬解码（NVDEC/NVENC）以及向下兼容的游戏图形接口支持。

命令执行完后，**先不要急着重启**，直接进入第 3 步。

## 3. 补全内核参数

继续在终端执行以下命令，修改内核启动参数：

```bash
rpm-ostree kargs --append=rd.driver.blacklist=nouveau --append=modprobe.blacklist=nouveau --append=nvidia-drm.modeset=1
```

这样做可以彻底镇住系统自带的开源驱动，防止一些诡异的事发生。包括但不限于系统自带软件打不开。

## 4. 重启系统

当驱动安装和内核参数都配置完毕后，重启电脑让新镜像生效：

```bash
systemctl reboot
```

至此驱动就安装完成了。
