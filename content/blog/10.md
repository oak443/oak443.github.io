### 1. 安装

```
sudo dnf install git -y
sudo dnf install gh -y
```

### 2. 登录 GitHub

```
gh auth login
```

随后可以选择 HTTP 然后用 Web Browser 登录，这样比较方便。

### 3. 设置全局的邮箱和用户名

不知道为什么，git必须要设置这个，而且 IDE 不会引导你完成，只能自己来了。

```
git config --global user.name "oak443"
git config --global user.email "tyg8614@gmail.com"
```

oak443 是用户名， tyg8614@gmail.com 是邮箱
