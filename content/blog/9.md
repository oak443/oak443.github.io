### c-development （必须）
```bash
sudo dnf install @development-tools @c-development
sudo dnf install cmake clang-tools-extra
```
### Node.js（必须）
```bash
sudo dnf install nodejs npm
```
### Rust
```bash
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
```
```bash
cargo install wasm-pack
```
### Typescript
```bash
sudo npm install -g typescript
```
### Go-lang
```bash
sudo dnf install golang
```
### VS Code
先按照这个： https://code.visualstudio.com/docs/setup/linux

然后安装 `libsecret`，不然没法登录。
```bash
sudo dnf install libsecret
```
### Gemini CLI
```bash
sudo npm install -g @google/gemini-cli
```
### Zed
按照这个： https://zed.dev/docs/installation#linux
### ADB
```bash
sudo dnf install android-tools
```
