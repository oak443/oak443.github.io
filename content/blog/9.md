## c-development （必须）

```bash
sudo dnf install @development-tools @c-development
sudo dnf install cmake clang-tools-extra
```

## Node.js（必须）

```bash
sudo dnf install nodejs npm -y
```

## Rust

```bash
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
```

```bash
cargo install wasm-pack
```

## Typescript

```bash
sudo npm install -g typescript
```

## Go-lang

```bash
sudo dnf install golang -y
```

## VS Code

先按照这个： <https://code.visualstudio.com/docs/setup/linux>

然后安装 `libsecret`，不然没法登录。

```bash
sudo dnf install libsecret -y
```

## ADB

```bash
sudo dnf install android-tools -y
```

## Google Antigravity

```bash
sudo dnf copr enable n0va-bot/agy
sudo dnf install antigravity -y
sudo dnf install antigravity-cli -y
sudo dnf install antigravity-ide -y
```
