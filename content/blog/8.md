## 分析段（安卓思维）

### 范文赏析 1

坚持 **就是** 胜利。面对困难和险阻， **唯有** 坚持，勇于永不言弃，**才** 有可能成功。欧立西为了发明一种新药，坚持进行实验，一直实验了 606 次，才获得了成功，制出了以实验次数命名的新药 606 。新药问世的 **原因就是** 坚持。 **假如** 欧立西仅仅实验一次，失败后就撒手不干了，新药当然不会问世； **假如** 他实验了 605 次，失败了以后不再坚持，也绝不可能有新药的制成。 **正是因为** 他坚持到了最后，才获得了最后的胜利。

### 基本结构——『三层六步法』（重要）

#### 第一层：观点 + 过渡

&#9312;观点：我们应该 / A 是 B / A 不是 B

&#9313;过渡：这句话告诉我们 A / 只有 A 才能 B / 如果 A 就 B 

#### 第二层：事例 + 分析

&#9314;事例：(注意交代背景，多个事例句式整齐。)

&#9315;分析：正因为 A 才 B / 为什么 A 呢，这是因为 B，所以 C / 以上 A 表明了 B 

#### 第三层：假设 + 观点

&#9316;假设：试想 A / 假如 A / 倘若 A / 倘若 A 还会 B 吗？

&#9317;观点：可见 A / 总而言之 A / 因此 A / 所以 A

### 范文赏析 2

**我们不应该** 以欺骗的方式对待生活。 *（观点）* **如果** 你以欺骗的方式对待生活， **那么** 生活势必会让你原型毕露。 *（过渡）* 林妙可在奥运会上，看似声音洪亮，却是用假唱欺骗观众，曾经的荣耀光环化为乌有。 *（事例）* **这是因为** 曾经以欺骗的方式呈现在大家的面前， **才** 会有今天群众的冷嘲热讽。 *（分析）* **假使** 当年她没有以欺骗的方式出现在观众面前，也就没有今天他人的指责。 *（假设）* **可见** 以欺骗的方式面对生活，那么你的生活最终也会像泡沫一样，绚丽而短暂。 *（结论）*

### 范文赏析 3

**我们应该** 以乐观的心态对待生活。  *（观点）* **如果** 你以乐观的心态对待生活， **那么** 生活会让你更加幸福快乐。 *（过渡）* **如果** 面对人生困境，艰难险阻，能保持乐观的心态， **那么** 生活也会奖励你更多的幸福。 *（过渡）* 一位业务员被医院误诊癌症，自以为时日不多，于是改变心态。在公司不再争辩，保持和谐，成了领导重视，同事欢迎的好员工；在家中不再打骂，陪伴家人，一家人幸福美满，和乐融融。 *（事例）* **这是因为** 有了乐观的心态，让他改变了原有的处事风格， **才** 有了今天充实的生活。 *（分析）* **如果** 这位员工没有改变心态，乐观面对，他的生活 **就** 不会越来越好，他的人生也不会如盛开的鲜花尽情绽放。 *（假设）* **可见** 以乐观的心态对待生活，生活就会更加丰富多彩。*（结论）*

### 练习 # 1

#### 题目

```
阅读下面材料，根据要求作文。
寒冬，枝叶凋零的树木默默积蓄养分。春风一到，树木便抽芽吐绿，尽显蓬勃生机。这种“藏”与“生”的智慧，何尝不蕴含在人生成长，社会发展之中？
请以“春藏冬生”为题目，写一篇议论文。要求：论点明确，论据充实，论证合理；语言流畅，书写清晰。
```

#### 作答

```
冬藏是默默无闻，也是积蓄力量，为我们养精蓄锐提供充足条件。如果面对生活的困境，我们能够不急于求成，潜心学习，那么就能丰富自己的见识，打破困局，实现新的突破。你看中国科学家屠呦呦，西医的局限性让她在药物研发中屡屡碰壁，面对研发瓶颈，她专心投入中医的研究，最终实现了发明青蒿素的成就。这是因为默默无闻地积攒力量，才有了前无古人的成就。如果屠呦呦不能默默积蓄力量，不能充实自我，在研究青蒿素时，知难而退，她也不会实现救济天下患疟者的人生理想。可见面对生活的寒冬，我们应该积蓄力量，为春生提供足够的养分。
```
