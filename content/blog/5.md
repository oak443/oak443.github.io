你不会喜欢片的声音自动传到客厅电视，对吧。

为了永远不发生这样的事情，应该按下面的步骤，禁用网络发现！

## 1. 创建配置文件目录

```bash
mkdir -p ~/.config/pipewire/pipewire.conf.d
cd ~/.config/pipewire/pipewire.conf.d
```

## 2. 写入配置

把下面的配置写到当前目录的 `99-disable-network-sink.conf` 文件里，你应该创建这个文件。

```conf
context.properties = {
    # 禁用 AirPlay 发现 (通常是小米电视出现的来源)
    module.raop = false
    
    # 禁用 Zeroconf/Avahi 发现 (Chromecast/DNLA 等)
    module.zeroconf.discover = false
}
```

## 3. 重启音频服务

```bash
systemctl --user restart pipewire
```

然后音频设备列表里的小米电视就永远消失了，耶！

## 4. 反安装

你喜欢片的声音自动传到客厅电视，对吧。

按照下面的步骤，启用网络发现。 :-)

1. 删除碍事的配置 `rm ~/.config/pipewire/pipewire.conf.d/99-disable-network-sink.conf`
2. 重启音频服务 `systemctl --user restart pipewire`
