## 1. 安装 keyd

按照页面上的 Installation Instructions 操作即可。
<https://copr.fedorainfracloud.org/coprs/alternateved/keyd/>

## 2. 写一个配置文件

创建一个全局配置文件：

```bash
sudo mkdir -p /etc/keyd
sudo nano /etc/keyd/default.conf
```

在文件中输入以下内容（实现 **Caps Lock** 和 **Enter** 的完全交换）：

```toml
# Gemini 3 生成
[ids]
*

[main]
# 将 Caps Lock 映射为 Enter
capslock = enter
# 将 Enter 映射为 Caps Lock
enter = capslock
```

## 3. 启动服务

```bash
sudo systemctl enable --now keyd
```
